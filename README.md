📊 Customer Churn Analysis (EDA)
📌 Project Overview

This project focuses on analyzing customer churn behavior using Exploratory Data Analysis (EDA) techniques. The goal is to identify key factors that influence customer attrition and provide actionable insights to improve retention.

🎯 Objective
Understand patterns behind customer churn
Identify high-risk customer segments
Analyze the impact of services and demographics on churn
Provide data-driven recommendations for retention

📂 Dataset Description
The dataset contains customer-level information including:
Demographics: Gender, Senior Citizen
Services: Internet Service, Online Security, Tech Support, etc.
Account Info: Tenure, Charges
Target Variable: Churn (Yes/No)

🛠️ Tools & Technologies
Python
Pandas
Matplotlib
Seaborn
Jupyter Notebook

📊 Key Insights
📉 Overall Churn Rate: ~26.5% (1 in 4 customers leave)
👵 Senior Citizens: ~41% churn vs ~24% for non-seniors
🌐 Fiber Optic Users: Highest churn (~41–42%)
🔐 No Online Security: ~3× higher churn (~42% vs ~15%)
🛠️ No Tech Support: ~42% churn vs ~15% with support
📦 No Device Protection/Backup: ~39–40% churn
🎬 Streaming Services: Minimal impact on churn

📈 Visualizations
The project includes:
Count plots for categorical features
Stacked bar charts with percentage distribution
Subplot grids comparing multiple features with churn

📌 Example:

🔍 Key Findings
Customers without support and security services are 2–3 times more likely to churn
Fiber optic service users show the highest churn risk
Value-added services significantly improve retention
Entertainment services have limited impact on churn decisions

💡 Conclusion
Customer churn is primarily influenced by service quality, support, and security features rather than demographics.

✅ Recommendations:
Offer bundled plans with tech support & security
Improve fiber optic service quality and pricing
Target high-risk segments like senior citizens
